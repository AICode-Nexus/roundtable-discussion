# 关于圆桌论坛

## 这是什么

「圆桌论坛」是一个多期制的跨角色深度讨论平台。每一期围绕一个核心主题，邀请产品经理、设计师、前端工程师、后端工程师、测试工程师等不同岗位的嘉宾，展开真实、有碰撞的对话。

## 为什么做这件事

AI 正在深刻改变软件团队的协作方式。每个岗位都在经历变化，但大家往往只看到自己的视角。我们希望通过圆桌论坛的形式，让不同角色的人坐在一起，聊聊各自的真实感受、困惑和判断。

不追求标准答案，追求真实碰撞。

## 论坛形式

- **多期制**：每期一个主题，持续产出
- **多角色**：产品、设计、前端、后端、测试，五个视角交叉碰撞
- **结构化**：每期设置开场题、冲突题、收束题，层层递进
- **快问快答**：轻松环节穿插，拉近距离

## 内容呈现

每个议题以「结构化观点摘要」的形式呈现：
- 各嘉宾的核心观点
- 共识与分歧总结
- 金句收录

## 参与方式

如果你有想讨论的主题，或者想作为嘉宾参与，欢迎通过 [GitHub Issues](https://github.com/ydliu2/roundtable-discussion/issues) 提出。
