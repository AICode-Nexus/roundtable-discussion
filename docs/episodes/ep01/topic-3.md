# 冲突升级：产品经理用 AI 出了设计稿和代码，设计和前端怎么看？

## 议题背景

这不是假设，这是正在发生的事。

2025-2026 年，AI 工具链已经成熟到一个产品经理可以：用 ChatGPT 写 PRD → 用 v0.dev 生成 UI → 用 Cursor 写出可运行的代码 → 用 AI 测试工具跑基础用例。全程不需要设计师和前端工程师介入。

Maor Shlomo 一个人用 AI 开发 Base44 平台，6 个月内达到 25 万用户，无联合创始人、无员工。Pieter Levels 零员工运营十几个产品，月收入超过 30 万美元。

当"越界"变成常态，岗位边界还有意义吗？
