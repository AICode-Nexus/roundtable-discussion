# 第十三期：AI 时代 — 超级个体的崛起与团队的重构

## 主题

当 AI 工具越来越强大，一个人可以做过去一个团队才能完成的事。这是「超级个体」的崛起，还是团队协作模式的一次重构？产品、设计、前端、后端、测试——每个角色都在被重新定义。

## 嘉宾

| 角色 | 代号 | 视角 |
|------|------|------|
| 🎯 产品经理 | PM | 需求定义者，离 AI 最近的决策层 |
| 🎨 设计师 | Designer | 用户体验的守护者，创意与规范的平衡 |
| 💻 前端工程师 | FE | 用户界面的实现者，AI 生成代码的第一接收人 |
| ⚙️ 后端工程师 | BE | 系统架构的构建者，稳定性与效率的权衡 |
| 🧪 测试工程师 | QA | 质量的最后防线，AI 代码的信任裁判 |
